# homebrew-tap

Homebrew tap for `rwx`, a terminal-based utility for inspecting and modifying file and directory permissions, ownership, and other metadata.

## Installation

To install `rwx` from this tap, run:

```bash
brew install vncsmnl/tap/rwx
```

Or tap this repository first:

```bash
brew tap vncsmnl/tap
brew install rwx
```

## Update

When a new version tag is pushed to the main `rwx` repository, the formula is automatically updated here via GitHub Actions.
